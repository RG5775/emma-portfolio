Project Overview

This project analyses over 42 million user interactions from a multi-category eCommerce store for 7 months to uncover user behaviour. By performing funnel analysis, we identify behavioural indicators such as add-to-cart rates, session engagement, timing of visits, and product interest across purchase journey.

The study reveals where users drop off in the conversion funnel and what drives successful purchases. Segmentation of new versus returning users highlights how shopping intent and behaviours differ, enabling tailored strategies to improve outcomes for both segments.

Featured Analysis: Brand Conversion Optimization

📓 View Complete Analysis Notebook

Explore the full Jupyter notebook with all SQL queries, data explorations, and detailed analysis 

WITH brand_view_purchase AS (
    SELECT
        brand,
        user_session,
        MAX(CASE WHEN event_type = 'view' THEN 1 ELSE 0 END) AS viewed,
        MAX(CASE WHEN event_type = 'purchase' THEN 1 ELSE 0 END) AS purchased
    FROM cleaned_data
    GROUP BY brand, user_session
)

SELECT 
    brand,
    COUNT(user_session) FILTER (WHERE viewed = 1) AS product_view,
    ROUND(100.00 * COUNT(user_session) FILTER (WHERE purchased = 1) / 
          NULLIF(COUNT(user_session) FILTER (WHERE viewed = 1), 0), 2) AS view_to_purchase_conversion
FROM brand_view_purchase
GROUP BY brand
ORDER BY view_to_purchase_conversion DESC;

Analysis Insight: This CTE-based query analyzes conversion rates across 4,202 brands, identifying which brands drive the highest view-to-purchase conversions within the same user session.
