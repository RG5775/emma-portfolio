Project Overview

This project analyzes customer behavior data from a multi-category eCommerce store to understand shopping patterns, customer preferences, and purchasing trends. The analysis provides insights into customer segmentation, product performance, and user journey optimization.

Power BI Data Analysis Customer Analytics SQL Python Statistical Analysis

Key Features

  • Customer behavior segmentation analysis across multiple product categories
  • Purchase pattern identification and trend analysis
  • Customer journey mapping and conversion funnel analysis
  • Product performance metrics and category comparison
  • Interactive filters for demographic and behavioral exploration
  • Predictive modeling for customer lifetime value

Data Insights

  • Seasonal purchasing patterns and peak shopping periods
  • Customer retention rates and churn analysis
  • Cross-category shopping behavior and product affinity
  • Geographic distribution of customer base
  • Mobile vs desktop shopping preferences
  • Impact of promotions and discounts on purchase behavior
