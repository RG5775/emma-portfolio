Project Overview

This project provides a comprehensive analysis of Walmart sales data to uncover revenue patterns, product performance, and market trends across different regions and time periods. The analysis was performed using Power BI for interactive data visualization and insights.

Power BI Data Analysis Data Visualization SQL Excel

Key Features

  • Global Retail Sales Dashboard displays the overall sales performance of Walmart over time
  • Store Comparison Dashboard compares the sales performance of different stores
  • Custom interactive filters for user-driven exploration
